# Terraform Sample for AWS and Google Cloud

This repository contains sample Terraform configurations for deploying infrastructure on AWS and Google Cloud Platform (GCP). It serves as a reference for setting up basic resources on both cloud providers using Terraform.

## Contents

- `AWS/`: Contains Terraform configurations for AWS resources.
- `GoogleCloud/`: Contains Terraform configurations for GCP resources.

## Prerequisites

|ツール|バージョン|
|---|---|
| Terraform | v1.5.7 |
| AWS CLI | 最新版 |
| Google Cloud SDK | 最新版 |
